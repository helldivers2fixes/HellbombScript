Privacy Policy:

This program queries the IP-API. Their privacy policy can be found here: https://ip-api.com/docs/legal
This program downloads the official CPU-Z binary. It performs SHA-256 hash verification after downloading. The author of this program has received permission from the CPU-Z author to utilize their work for this script.
The CPU-Z privacy policy can be found here: https://valid.x86.fr/privacy-policy.html

Build process:

The release exe is built directly in the github action.
The release exe is then downloaded for verification, testing and uploaded to both Virus Total and Hybrid Analysis before being uploaded to the release page and published.
